# Digital Parking Permit System
**Course:** SER 216 — Software Enterprise  
**Student:** Khondamir Tuychiev

## Project Description
This project designs and tests a Digital Parking Permit System for a university. 
Students and staff can register vehicles, request permits, check parking zones, 
and report issues. Security staff can verify permits and log violations.

## Workflow
Vehicle Registration → Permit Request → Zone Check → Permit Approval → Parking Verification
